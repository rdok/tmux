### Building Habits

| Command                       | Info                                      | 
| -------------------------     |:-----------------------------------------:|
| `tmux new -s basic`           | Created a session named 'basic'.          |
| `<Ctrl-b>{command}`           | Execute command in tmux. E.g. `<Ctrl-b>t` |
| `<Ctrl-b>d`                   | Detach from session                       |
